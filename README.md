# Study Area Map of Rajshahi Development Authority (RDA)

![RDA Study Area Map](RDA_Study_Area_Map.jpg)

## Overview

This repository presents the **study area map of the Rajshahi Development Authority (RDA), Bangladesh**. The map was prepared to define and visualize the spatial boundary of the study area used in GIS-based analysis.

The map provides the geographical context for research related to **Land Use Land Cover (LULC) analysis, urban expansion, and spatial planning** within the RDA region.

---

## Study Area

The **Rajshahi Development Authority (RDA)** is responsible for planning and managing urban development in the Rajshahi metropolitan region of Bangladesh.

The study area map illustrates the spatial extent of the RDA planning area along with its geographic location. It serves as the base reference for spatial analysis conducted within this region.

---

## Methodology

The study area map was prepared using **GIS-based cartographic techniques**. The main steps included:

1. Collecting spatial boundary data of the RDA planning area.
2. Processing and organizing spatial layers in a GIS environment.
3. Creating map layout including:
   - Study area boundary
   - Geographic coordinates
   - Scale bar
   - North arrow
   - Map legend
4. Final cartographic visualization for presentation and analysis.
